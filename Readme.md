# Counter Web App

An interactive counter application built with **React** and **Vite**, featuring a clean, responsive design powered by **Tailwind CSS**.

## Features

- **Interactive Controls**: Seamlessly increment, decrement, and reset the counter state.
- **Smart Validation**: Logic prevents negative numbers and provides visual feedback or message when the lower limit is hit.
- **User Personalization**: A welcoming interface that allows users to set a display name.
- **Responsive UI**: Optimized for various screen sizes with a modern, minimalistic aesthetic.

## Getting Started

Follow these steps to run the application locally:

1. **Install Dependencies**

    ```bash
    npm install
    ```

2. **Run Development Server**

    ```bash
    npm run dev
    ```
